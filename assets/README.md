# 资源文件说明

## 📂 目录结构

```
assets/
├── fonts/                 # 字体文件
│   ├── PlayfairDisplay-*.ttf
│   ├── Inter-*.ttf
│   └── README.md
├── icon.png              # 应用图标 (1024x1024)
├── splash.png            # 启动画面
├── adaptive-icon.png     # Android 自适应图标
├── favicon.png           # Web 图标
└── placeholder.png       # 占位图片
```

## 🖼️ 图标文件要求

### icon.png

- **尺寸**: 1024x1024 像素
- **格式**: PNG
- **用途**: iOS 和 Android 应用图标
- **设计建议**: 简洁、可识别、符合应用主题

### splash.png

- **尺寸**: 1284x2778 像素 (iPhone 12 Pro Max)
- **格式**: PNG
- **用途**: 应用启动画面
- **设计建议**: 与应用图标保持一致的视觉风格

### adaptive-icon.png

- **尺寸**: 1024x1024 像素
- **格式**: PNG
- **用途**: Android 自适应图标前景
- **注意**: 考虑不同形状的遮罩效果

### favicon.png

- **尺寸**: 32x32 像素
- **格式**: PNG
- **用途**: Web 版本图标
- **设计**: 应用图标的简化版本

## 🎨 设计规范

### 色彩主题

- **主色**: #000000 (黑色)
- **辅色**: #FFFFFF (白色)
- **强调色**: #C7A27C (香槟金)

### 设计风格

- **极简主义**: 简洁、优雅
- **高对比度**: 黑白为主
- **时尚感**: 符合 Vogue 风格

## 🔄 替换步骤

1. **准备设计文件**

   - 使用 Sketch、Figma 或 Photoshop 创建
   - 确保符合尺寸要求
   - 导出为 PNG 格式

2. **替换文件**

   - 保持文件名不变
   - 直接覆盖现有占位符文件

3. **测试验证**
   - 运行 `npm start` 启动应用
   - 检查图标显示效果
   - 在不同设备上测试

## 📱 平台特定要求

### iOS

- 图标会自动添加圆角
- 不需要预先添加圆角效果
- 建议使用饱和度较高的颜色

### Android

- 支持自适应图标
- 会根据系统主题调整形状
- 前景图标应留有足够边距

## 🚀 优化建议

1. **文件大小优化**

   - 使用 TinyPNG 等工具压缩
   - 保持质量的同时减小文件大小

2. **多分辨率支持**

   - 考虑不同设备的显示密度
   - Expo 会自动处理缩放

3. **品牌一致性**
   - 所有图标保持统一风格
   - 与应用内设计保持一致

## ⚠️ 注意事项

- 当前文件为占位符，仅用于开发测试
- 生产环境必须替换为实际设计的图标
- 确保所有图标都有适当的版权许可
- 遵循 App Store 和 Google Play 的图标设计指南
